문제 정의

백석대학교 Smart_it 출품작으로 반려견 케어 어플리케이션을 만들었습니다.

설계 및 구현

Exaone3.5를 파인튜닝하여 LLM 모델을 구축하였으며, 역번역을 통해 사용자의 Input을 늘려 정확한 답변을 하도록 개선하였습니다.

# 1. 사용할 모델 이름
LOCAL_MODEL_NAME = "exaone3.5"

# 2. Ollama 주소 설정
default_url = "http://localhost:11434/v1"
OLLAMA_URL = os.getenv("OLLAMA_URL", default_url)

# Ollama 클라이언트 초기화
client = OpenAI(
    base_url=OLLAMA_URL,
    api_key="ollama"
)

# 3. 장치 자동 설정
device = "cuda" if torch.cuda.is_available() else "cpu"

결과 및 성능

한정된 데이터로 특정 답변에서 어려움을 보임. 예를 들어, 피부병 관련 검색을 하면 질문자의 견종은 말티즈여도 포메라니안에 대한 답변이 나오는 문제가 있었습니다.
