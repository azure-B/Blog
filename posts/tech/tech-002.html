#CNN #Convolution #Pooling #Computer Vision

CNN은 이미지의 공간적 지역성(Spatial Locality)을 활용해 파라미터 수를 대폭 줄이는 딥러닝 구조입니다. IPCG 랩에서 다루는 이미지 처리의 핵심 기반이기도 합니다.

1. 합성곱(Convolution) 연산

입력 이미지에 작은 필터(커널)를 슬라이딩하며 내적을 계산합니다. 동일한 필터가 이미지 전체를 훑으므로 가중치 공유(Weight Sharing)가 이루어집니다.

Feature Map(i,j) = Σₘ Σₙ Input(i+m, j+n) · Filter(m,n) + b

2. 출력 크기 계산

# 출력 크기 계산 공식
# Input: H×W, Filter: F×F, Padding: P, Stride: S

output_h = (H + 2P - F) // S + 1
output_w = (W + 2P - F) // S + 1

# 예: 28×28 입력, 3×3 필터, P=1, S=1
# → (28 + 2 - 3) // 1 + 1 = 28  (same padding)

3. 풀링(Pooling)

Max Pooling은 특징 맵에서 최댓값만 추출해 공간 크기를 줄이고, 위치 불변성(Translation Invariance)을 제공합니다.

⚡ Same vs Valid Padding
Same: 출력 크기 = 입력 크기 (P = F//2)
Valid: 패딩 없음, 출력이 작아짐 (P = 0)

4. PyTorch 구현 예시

import torch.nn as nn

class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 32, kernel_size=3, padding=1)
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.pool  = nn.MaxPool2d(2, 2)
        self.fc    = nn.Linear(64 * 7 * 7, num_classes)
        self.relu  = nn.ReLU()

    def forward(self, x):
        x = self.pool(self.relu(self.conv1(x)))  # 28→14
        x = self.pool(self.relu(self.conv2(x)))  # 14→7
        x = x.view(x.size(0), -1)
        return self.fc(x)
