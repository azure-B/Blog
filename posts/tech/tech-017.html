1. Machine Learning?

우리가 일일이 프로그래밍을 하지 않고도,
프로그램이 데이터를 보고학습하여 무언가를 배우는 프로그램.

1-1. Supervised Learning ( 지도 학습 )

Labeling되어진 Data를 통해 학습을 하는 것.
즉, 정답을 가진 데이터를 가지고 학습을 하는 것이다.

1-2. Unsupervised Learning

우리가 Labeling을 할 수 없는 데이터를 통해 학습 하는 것.
Google의 news grouping 데이터들이 예시이다.
