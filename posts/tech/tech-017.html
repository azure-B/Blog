2. Linear Regression

Lec 02. YOUTUBE LINK

어떤 Linear한 Model이 가지고 있는 데이터에 맞을 것이다.
가설을 세우며 진행 하는 것을 말한다.

예를들어 공부 시간이 성적시간에 비례하듯이, 많은 경우에 쓰인다.

2-1. Hypothesis ( 가설 )

어떤 것이 우리에게 잘 맞는 데이터일까?
그 선을 찾는 것이 학습이다.

가설
$H(x) = Wx + b$

Linear Regression의 경우, 데이터를 연결 했을 때 어떤 선형이 나온다.
이 경우 우리의 가설은 $H(x) = Wx + b$의 일차방정식 선형을 띄게 된다.

2-2. Which Hypothesis is better?

어떤 선이 우리 데이터에 가장 좋은 값인지? 찾아야 하기 때문에,
가설과 데이터의 거리를 계산한다.

2-3. Cost(Loss) function

여기서 가설과 실제 데이터가 얼마나 다른지 표시해주는 것이 Cost function이다.


Cost
$H(x) - y$

즉 실제 값에서, 가설의 값을 빼주는 것 이다.
허나 이 경우 거리의 값이 음수가 되는 경우가 있기에


Cost
$(H(x) - y)^2$

자승을 취하여 사용한다.
이 경우 실제 데이터와 가설의 거리가 멀 경우, 큰 페널티를 줄 수도 있다.


Cost
$cost = \frac{1}{m} \sum_{i=1}^{m} (H(x^{(i)}) - y^{(i)})^2$

변수 정의 $m$ = 데이터의 갯수
$H(x^(i))$ = 모델의 입력 값에 따른 예측 출력 값
$y^(i)$ = 실제 결과

모든 입력 값에 대한 예측값과 실제값의 차이를 자승하여 더하고,
평균을 내는 것을 Cost Function이라 한다.


가설
$H(x) = Wx + b$

이 때 앞서 배운 Linear Regression의 함수를 자연스레 $H(x)$에 대입하면,


Cost
$cost(W,b) = \frac{1}{m} \sum_{i=1}^{m} (H(x^{(i)}) - y^{(i)})^2$

위와 같이 표현 해줄 수 있다.

여기서 Linear Regression의 숙제는 W와 b를 가장 작게 만드는 것이고,
이 것이 곧 학습이다.


Goal
$minimize cost(W,b)$
