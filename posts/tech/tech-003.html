"Attention is All You Need" (Vaswani et al., 2017) 이후 Transformer는 NLP를 넘어 Vision, Audio 전 영역의 표준이 되었습니다. 그 핵심인 Self-Attention을 처음부터 뜯어봅니다.

1. Query, Key, Value

입력 시퀀스의 각 토큰은 세 가지 벡터로 선형 변환됩니다.

Q = XWQ, K = XWK, V = XWV

직관적으로: Q는 "내가 뭘 찾고 있나?", K는 "나는 무엇인가?", V는 "실제 정보"입니다.

2. Scaled Dot-Product Attention

Attention(Q, K, V) = softmax( QKᵀ / √dₖ ) · V

√dₖ로 스케일링하는 이유: 차원이 커질수록 내적이 커져 소프트맥스가 포화(Saturation)되기 때문입니다.

3. NumPy 구현

import numpy as np

def scaled_dot_product_attention(Q, K, V, mask=None):
    d_k = Q.shape[-1]
    scores = Q @ K.T / np.sqrt(d_k)

    if mask is not None:
        scores = scores + (mask * -1e9)

    weights = np.exp(scores) / np.sum(np.exp(scores), axis=-1, keepdims=True)
    return weights @ V, weights
