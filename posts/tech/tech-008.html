1. KL Divergence

KL Divergence 이전 Entropy 사전지식 필수
엔트로피와 최대우도추정

이제 두 개의 분포 실제 분포$P$와 예측 분포$Q$가 있다고 가정한다.

교차 엔트로피
$H(P, Q) = \mathbb{E}_{x \sim P} [-\log Q(x)] = -\sum_x P(x) \log Q(x)$
엔트로피
$H(P) = E_{x \sim P}[I(x)]$
↓ 교차 엔트로피 - 엔트로피
KL Divergence
$D_{KL}(P \parallel Q) = H(P, Q) - H(P)$
↓
KL Divergence
$= \left( -\sum P(x) \log Q(x) \right) - \left( -\sum P(x) \log P(x) \right)$
↓
KL Divergence
$= \sum_x P(x) \left( \log P(x) - \log Q(x) \right)$
↓
KL Divergence ( 이산 분포 )
$= \sum_x P(x) \log \frac{P(x)}{Q(x)}$

KL Divergence ( 연속 분포
$\int P(x) \log\left(\frac{P(x)}{Q(x)}\right) dx$

KL Divergence는 교차 엔트로피 - 엔트로피 이며
잘못된 분포 $Q$를 사용해서 발생하는 추가적인 정보량이다.

결론은 두 확률 분포간의 차이를 의미하며
수식을 설명하면, 내가 P대신 Q를 선택했기 때문에 낭비된 정보량이다.

1-1 성질

1-1-1 왜 거리가 아닌가?

수학에서 거리 즉 Distance라고 하려면 대칭성이 있어야 한다. ($A \to B$ = $B \to A$)


KL Divergence -> Q(x)에 대해 미분
$\frac{\partial KL}{\partial Q(x)} = \frac{\partial}{\partial Q(x)} \left[ -P(x) \log Q(x) \right] = - \frac{P(x)}{Q(x)}$

하지만, $KL(P \parallel Q)$에서 P가 있는 곳은 반드시 Q가 있어야 한다.
그리고 $KL(Q \parallel P)$에서 Q가 있는 곳은 반드시 P가 있어야 한다.

만약 $P$가 0보다 클 때 $Q$가 0에 가까워 진다면, 값이 무한대로 발산한다.
즉 실제로 일어날 확률 $P$가 0이 아닌데, $Q$가 0에 가까워진다면, $- \frac{P(x)}{Q(x)} \to -\infty$ 가 되어
모델이 $Q = 0$이 되어지는 상황을 피하려 큰 기울기 값을 갖게 된다.

반대로 $Q$가 0보다 클 때 $P$가 0에 가까워 진다면, 값이 무한대로 발산한다.
즉 예측한 확률 $Q$가 0이 아닌데, 실제 확률$P$가 0에 가까워진다면, $-\frac{P(x)}{Q(x)} \to \infty$ 가 되어
모델이 $Q = 0$이 되어지려 큰 기울기 값을 갖게 된다.

$P_1, P_2$가 있다면, $Q$가 $P_1=P_2=0$이라는 중간 지점을 건드리지 않으려
$P_1, P_2$중 하나의 값만을 참조하게 되므로 모드붕괴가 일어난다.

$\log(x)$를 미분하는데 왜 $\ln(x)$가 아닌가? 최대 우도함수, 엔트로피, 교차 엔트로피 모두 밑이 $e$인 $\log$이다.

1-2. KL Divergence는 왜 항상 0 이상인가?


$-D_{KL}$의 형태
$-D_{KL}(P \parallel Q) = -\sum_x P(x) \log \frac{P(x)}{Q(x)} = \sum_x P(x) \log \frac{Q(x)}{P(x)}$
↓ 기댓값의 정의 적용
기댓값의 정의
$\sum_x P(x) \log \frac{Q(x)}{P(x)} = \mathbb{E}_{x \sim P} \left[ \log \frac{Q(x)}{P(x)} \right]$
↓ 젠센의 부등식 적용
젠센의 부등식
$\mathbb{E} \left[ \log \frac{Q(x)}{P(x)} \right] \le \log \left( \mathbb{E} \left[ \frac{Q(x)}{P(x)} \right] \right)$
↓ 기댓값을 풀어 정리
정리
$\log \left( \sum_x P(x) \frac{Q(x)}{P(x)} \right) = \log \left( \sum_x Q(x) \right)$
↓ 확률의 총합은 1이므로
정리
$\log(1) = 0$

따라서, $-D_{KL} \le 0$ 이므로 $D_{KL}(P \parallel Q) \ge 0$ 임이 증명된다.

젠센의 부등식 아래로 볼록한 함수 일 때
$f(\mathbb{E}[X]) \le \mathbb{E}[f(X)]$
평균을 먼저 구해서 함수에 넣은 값보다, 함수에 먼저 넣고 그 결과들을 평균 낸 값이 더 크거나 같다.

위로 오목한 함수 일 때
$f(\mathbb{E}[X]) \ge \mathbb{E}[f(X)]$
평균의 함수값이 함수값의 평균보다 크거나 같다.

결론적으로 KL Divergence는 $Q$가 $P$를 얼마나 효율적으로 설명하는가를 측정하는 지표이다.
단순히 오차를 줄이는 것을 넘어, 모델의 학습 방향을 결정짓는 나침반 역할을 한다.
