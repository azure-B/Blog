로지스틱 회귀(Logistic Regression) 완전 분석

이승빈 2025.06.10
#Logistic Regression #MLE #Gradient Descent #Classification

선형 회귀가 연속값을 예측한다면, 로지스틱 회귀는 이진 분류(Binary Classification) 문제를 확률로 풀어냅니다. 이름에 '회귀'가 붙어 있지만 실질적으로는 분류 알고리즘입니다.

1. 시그모이드 함수

로지스틱 회귀의 핵심은 선형 결합 값을 0~1 사이의 확률로 변환하는 시그모이드(Sigmoid) 함수입니다.

σ(z) = 1 / (1 + e⁻ᶻ) where z = wᵀx + b

출력값이 0.5 이상이면 클래스 1, 미만이면 클래스 0으로 분류합니다. 미분 시 σ'(z) = σ(z)(1 - σ(z))라는 깔끔한 형태가 나와 역전파에 유리합니다.

2. 손실 함수: Binary Cross-Entropy

로지스틱 회귀는 최대 우도 추정(MLE)으로 파라미터를 학습합니다. 이를 최소화 문제로 바꾸면 Binary Cross-Entropy Loss가 도출됩니다.

L(w,b) = -1/N · Σ [ yᵢ log(ŷᵢ) + (1−yᵢ) log(1−ŷᵢ) ]

3. 경사하강법으로 파라미터 업데이트

import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def logistic_regression(X, y, lr=0.01, epochs=1000):
    m, n = X.shape
    w = np.zeros(n)
    b = 0

    for _ in range(epochs):
        z = X @ w + b
        pred = sigmoid(z)

        dw = (1/m) * X.T @ (pred - y)
        db = (1/m) * np.sum(pred - y)

        w -= lr * dw
        b -= lr * db

    return w, b
⚡ 핵심 포인트:
로지스틱 회귀의 결정 경계(Decision Boundary)는 선형입니다. 비선형 경계가 필요하다면 특징 공학(Feature Engineering)이나 커널 SVM, 신경망을 고려하세요.

4. 정리

  • 시그모이드로 선형값 → 확률 변환
  • BCE Loss를 MLE 관점에서 유도 가능
  • 경사하강법으로 w, b 업데이트
  • 선형 결정 경계를 가진 이진 분류기
