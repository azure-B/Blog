#Machine Learning

1. Type of Supervised Learning

1-1. Regression ( 회귀 )

회귀는 통계학과 머신러닝에서 하나 이상의
독립 변수(입력)과 연속적인 종속 변수(출력) 사이의 수학적 관계를 추정하고 모델링하는 기법이다.

데이터들의 패턴을 파악하여 연속된 수치를 예측하는 것이 회귀의 핵심이다.
예를들어, 집의 평수가 지하철역과의 거리 등을 바탕으로 '아파트 가격'을 예측하는 문제이다.

1-2. binary Classification ( 이진 분류 )

이진 분류는 데이터가 두 가지 범주 중 하나를 판별하는 지도 학습 문제이다.
입력 데이터를 바탕으로 예 또는 아니오 라는 두 가지 결론 중 하나를 내리도록 학습시킨다.

1-3. Multi-label Classification ( 다중 레이블 분류 )

하나의 데이터가 여러 개의 정답을 가질 수 있는 문제를 해결하는 기법이다.
가장 직관적인 예시는 영화 장르 분류나 이미지 태깅이다.
