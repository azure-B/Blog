처음으로 Dacon 이미지 분류 대회에 참가했습니다. 결과보다 배운 것에 초점을 맞춰 기록합니다.

접근 방법

IPCG 랩에서 쌓은 CNN 지식을 바탕으로, EfficientNet-B3를 백본으로 선택하고 마지막 분류 레이어만 학습시키는 파인튜닝 전략을 사용했습니다.

from torchvision.models import efficientnet_b3
import torch.nn as nn

model = efficientnet_b3(pretrained=True)

num_classes = 88
model.classifier[1] = nn.Linear(
    model.classifier[1].in_features,
    num_classes
)

# 백본 동결 (초기 학습 단계)
for param in model.features.parameters():
    param.requires_grad = False

실험 결과

실험 요약:
① EfficientNet-B3 Fine-tune: Acc 84.2%
② + TTA(Test Time Augmentation): 86.1%
③ + B5 앙상블: 88.7% (최종 제출)

회고

  • 데이터 불균형 처리를 처음부터 했더라면 더 좋은 결과가 나왔을 것
  • 앙상블 효과가 생각보다 크다는 것을 실감함
  • 다음 대회에는 Focal Loss부터 적용해볼 예정
