문제 정의

IPCG 랩 세미나에서 "화면을 터치하지 않고 발표 슬라이드를 제어할 수 있을까?" 라는 아이디어에서 출발했습니다. 기존 솔루션은 무거운 모델을 사용해 지연이 발생했기에, 경량 실시간 시스템을 직접 구현하기로 했습니다.

설계 및 구현

MediaPipe Hands로 21개의 손 랜드마크를 추출한 뒤, 좌표를 정규화해 경량 MLP 분류기에 입력합니다.

import mediapipe as mp

mp_hands = mp.solutions.hands

def extract_landmarks(frame):
    with mp_hands.Hands(
        static_image_mode=False,
        max_num_hands=1,
        min_detection_confidence=0.7
    ) as hands:
        result = hands.process(frame)
        if result.multi_hand_landmarks:
            lm = result.multi_hand_landmarks[0].landmark
            return [(p.x, p.y) for p in lm]
    return None

결과 및 성능

10가지 제스처를 평균 96.4% 정확도로 분류했습니다. 추론 속도는 RTX 3060에서 약 2ms / 프레임으로, 웹캠 30fps에서 실시간 동작을 확인했습니다.

"손 좌표 정규화 시 손목을 원점으로 설정하고 손바닥 크기로 나누는 방식이 회전과 거리 변화에 강건함을 보였습니다."
