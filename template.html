/* ① 블록 수식 — .formula div 안에 $$...$$ */

  $$\sigma(z) = \frac{1}{1 + e^{-z}}$$
</div>

/* ② 본문 인라인 수식 — $...$ */
<p>
  시그모이드 함수 $\sigma(z)$는 출력을 $[0, 1]$ 범위로 변환합니다.
</p>

/* ③ 여러 줄 수식 — \\ 로 줄바꿈, aligned 환경 */
<div class="formula">
$$\begin{aligned}
  \frac{\partial \mathcal{L}}{\partial w} &= \frac{1}{N} X^\top (\hat{y} - y) \\
  \frac{\partial \mathcal{L}}{\partial b} &= \frac{1}{N} \sum_{i=1}^{N} (\hat{y}_i - y_i)
\end{aligned}$$
</div>

/* ④ 자주 쓰는 LaTeX 기호 모음 */
  분수:      \frac{분자}{분모}         →  ½ 형태
  합:        \sum_{i=1}^{N}            →  Σ
  곱:        \prod_{i=1}^{N}           →  Π
  루트:      \sqrt{x},  \sqrt[3]{x}   →  √
  벡터:      \mathbf{w}, \vec{v}       →  볼드/화살표
  전치:      W^\top                    →  Wᵀ
  정규분포:  \mathcal{N}(\mu, \sigma^2)
  편미분:    \frac{\partial L}{\partial w}
  기댓값:    \mathbb{E}[X]
  절댓값:    \left| x \right|
